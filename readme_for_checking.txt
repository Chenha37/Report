בתיקייה ישנם מספר קבצים:
הקובץ שמכיל את האלגוריתם - best_three_players_algorithm.ipynb
כל הקבצים המצורפים כאן צריכים להיות בתיקייה עם ההרצה (האלגוריתם משתמש בהם).

הקובץ predictor.ipynb מכיל את המחברת איתה הרצנו ובדקנו את הרגרסורים השונים.
הקובץ csv_merge.py מכיל את כל קטעי הקוד שכתבנו איתם יצרנו את הקבצים מקובץ הdatabase המקורי

כל הקבצים הנוספים נוצרו באמצעות קטעי הקוד הכתובים בcsv_merge.py 
בהם השתמשנו בריצה של האלגוריתם וגם ברגרסורים בשלב הראשון.
